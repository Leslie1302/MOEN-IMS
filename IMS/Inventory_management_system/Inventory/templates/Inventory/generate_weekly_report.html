❓ Help & Tips

Email Configuration

Make sure your Django email settings are configured in settings.py:

  • EMAIL_BACKEND - Email backend to use
  • EMAIL_HOST - SMTP server address
  • EMAIL_PORT - SMTP port (usually 587 or 465)
  • EMAIL_HOST_USER - SMTP username
  • EMAIL_HOST_PASSWORD - SMTP password
  • EMAIL_USE_TLS - Whether to use TLS
  • DEFAULT_FROM_EMAIL - Sender email address

Default Recipients

Set default recipients in settings.py:

WEEKLY_REPORT_RECIPIENTS = ['manager@example.com', 'supervisor@example.com']

CLI Usage

You can also generate reports from the command line:

python manage.py generate_weekly_report --dry-run
python manage.py generate_weekly_report --days 7 --recipients email@example.com

Scheduling

To send reports automatically every week, set up a cron job:

0 9 * * 1 cd /path/to/project && python manage.py generate_weekly_report

(This runs every Monday at 9:00 AM)
