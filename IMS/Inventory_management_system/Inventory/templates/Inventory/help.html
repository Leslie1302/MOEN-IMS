Help Center

Getting Started

Welcome to the Inventory Management System help center. Here you'll find information on how to use the system.

Navigation

  • Dashboard: View an overview of your inventory and recent activities.
  • Request Material: Submit requests for new materials.
  • Material Orders: Track the status of your material orders.
  • Bill of Quantity: View detailed quantity information.
  • Low Inventory: Check items that are low in stock.
  • Heatmap Report: Visualize material usage and trends.

Frequently Asked Questions

How do I request new materials?

Navigate to the "Request Material" page and fill out the form with the required details. Submit the form to create a new material request.

How can I check the status of my orders?

Go to the "Material Orders" page to view the status of all your orders. You can filter and search for specific orders as needed.

How do I use the table features?

All tables in the system now support the following features:

  • Sorting: Click on any column header to sort by that column.
  • Filtering: Use the search box to filter results or click on the filter icon in column headers for advanced filtering.
  • Pagination: Navigate through multiple pages of results using the pagination controls.
  • Export: Export table data to CSV, Excel, PDF, or print directly from the table.

What should I do if I encounter an issue?

If you experience any problems, please check the following before contacting support:

  • Ensure you have the latest version of your web browser
  • Clear your browser cache if tables are not displaying correctly
  • Verify your internet connection

If the issue persists, please contact your system administrator for assistance.
