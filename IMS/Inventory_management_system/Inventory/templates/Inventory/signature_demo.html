🎨 Customization

You can customize the stamp appearance by modifying signature_stamp.css

Available CSS Classes:

  • .digital-signature-stamp - Main stamp container
  • .stamp-border - Border styling
  • .stamp-header - "AUTHORIZED SIGNATURE" text
  • .stamp-username - Username display
  • .stamp-role - User role/title
  • .stamp-footer - Year and ID section

Color Variants:

Add these classes to .stamp-border:

  • stamp-blue - Blue border
  • stamp-green - Green border
  • stamp-purple - Purple border
  • Default - Red border
