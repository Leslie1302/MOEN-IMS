ℹ️ Instructions

Upload an Excel file (.xlsx or .xls) with the following columns:

  • username - Unique username for login (required)
  • name - Full name of the user (e.g., "John Doe")
  • email - Email address (required)

Note: Passwords will be generated automatically and can be sent to users via email.

📋 Excel File Format Example:
username name email
jdoe John Doe john.doe@example.com
asmith Alice Smith alice.smith@example.com
bwilson Bob Wilson bob.wilson@example.com

⚠️ Important Notes

  • Users with duplicate usernames or emails will be skipped automatically
  • Passwords are 12 characters long (letters and numbers)
  • If "Send welcome email" is checked, users will receive their credentials via email
  • All users will be created with standard user permissions unless assigned to a group
