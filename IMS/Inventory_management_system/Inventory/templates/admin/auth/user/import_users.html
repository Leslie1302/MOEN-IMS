📋 Excel File Requirements

  • File Format: Excel (.xlsx or .xls)
  • Required Columns:
    • username - Unique username for login

    • email - Valid email address (must be unique)
  • File Size: Maximum 10MB
  • Row Limit: Maximum 1000 users per import
  • Data Validation: Duplicate usernames and emails will be rejected

🔐 Password Generation

The system will automatically generate secure passwords for each user with the following characteristics:

  • 12 characters long
  • Contains uppercase and lowercase letters
  • Contains numbers and special characters
  • Cryptographically secure random generation

Important: Generated passwords will be displayed after import. Make sure to save them securely!
